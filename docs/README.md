# Документация Nyx

Здесь собрана вся проектная документация. Её нужно держать в актуальном состоянии при каждом изменении кода (см. правило в `.cursor/rules/Nyx-development.mdc`).

| Документ | Для кого | Содержание |

## Текущая стадия

**Прототип транспорта (v0.1).** Два узла могут соединиться через rendezvous и обменяться ping. GUI (`nyx-app`) запланирован с фазы 1; целевой вид — Telegram-like, см. [GUI.md](GUI.md).

## Связь с кодом

Описание публичных функций дублируется в комментариях к заголовкам `include/nyx/*.hpp`. При изменении API сначала обновляют код и комментарий, затем соответствующий раздел в APPLICATION.md или ARCHITECTURE.md.
